What is Topology?

Topology is the representation of all the network elements in a network.

What is the TAF Topology Handler?

The TAF Topology Handler is an API which allows you to get this topology from the OSS.

How to get started

The Topology handler is used by the OSS Handler and so this handler should be referenced and used.
