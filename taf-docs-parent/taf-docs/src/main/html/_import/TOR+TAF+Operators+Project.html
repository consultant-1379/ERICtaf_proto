Introduction

Welcome to the TAF TOR Operators confluence pages. The goal of these pages is to introduce the project, explain why the project is needed and to outline the ways of working.

What is the TAF TOR Operators project

In it's simplest terms the TAF TOR Operators project is a maven project, it has a parent (the integration pom) and it has dependencies i.e. all operators created by the TOR development teams. The purpose of the project is to create a method for operators created by a team for the purpose of automated testing to be shared with other teams involved in the development of TOR.
